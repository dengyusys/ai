# 个人知识库导航

## 概述
这是我的个人知识管理系统核心存储区域。通过结构化目录组织不同类型的知识，便于检索和关联。

## 目录结构说明

### 1. 领域知识 (areas/)
系统性的知识，按主题领域分类：
- **technology/** - 技术相关：编程、工具、架构等
- **work/** - 工作相关：项目、流程、职业发展
- **personal/** - 个人生活：健康、家庭、兴趣
- **learning/** - 学习笔记：课程、技能、研究

### 2. 日常笔记 (notes/)
临时性记录和灵感：
- **daily/** - 日报、周报、日常记录
- **meeting/** - 会议记录、讨论要点
- **ideas/** - 灵感想法、创意记录

### 3. 参考资料 (references/)
外部资料整理：
- **articles/** - 文章摘要、读后感
- **books/** - 读书笔记、书摘
- **links/** - 有用链接、资源收藏

### 4. 项目文档 (projects/)
项目相关材料：
- **current/** - 进行中项目的文档
- **archived/** - 已完成项目的归档

### 5. 知识模板 (templates/)
标准化知识记录格式：
- **note-templates/** - 笔记结构模板
- **review-templates/** - 复习回顾模板
- **analysis-templates/** - 分析框架模板

## 使用指南
1. 新知识根据性质放入对应目录
2. 使用模板确保记录一致性
3. 建立知识点之间的关联链接
4. 定期回顾和整理
