# 问答模式模板

## 概述
标准化的问答处理流程，确保问题分析和回答的一致性。

## 基础问答模式

### 1. 定义类问题（什么是X？）
**模板结构：**
```
### 问题：什么是[概念]？

**核心定义**
- 基本概念：[简明定义]
- 关键特征：[主要特点]
- 相关术语：[关联概念]

**详细解释**
1. 起源背景：[发展历史/来源]
2. 工作原理：[机制说明]
3. 应用场景：[使用情况]

**知识关联**
- 相关领域：[关联知识领域]
- 学习资源：[进一步学习材料]
- 实践建议：[应用建议]
```

### 2. 方法类问题（如何做X？）
**模板结构：**
```
### 问题：如何[达成目标]？

**步骤指南**
1. 准备阶段：[前提条件/准备工作]
2. 实施步骤：[具体操作步骤]
3. 验证阶段：[检查/测试方法]

**注意事项**
- 常见错误：[避免的问题]
- 优化建议：[改进方法]
- 时间预估：[大致耗时]

**资源需求**
- 工具材料：[所需工具]
- 技能要求：[必要技能]
- 参考资料：[相关文档]
```

### 3. 比较类问题（X vs Y）
**模板结构：**
```
### 问题：[A]和[B]的比较

**对比维度**
| 维度 | [A] | [B] | 适用场景 |
|------|-----|-----|----------|
| 性能 | | | |
| 成本 | | | |
| 易用性 | | | |
| 扩展性 | | | |

**选择建议**
- 推荐[A]的情况：[具体场景]
- 推荐[B]的情况：[具体场景]
- 折中方案：[平衡选择]

**迁移考虑**
- 从A到B：[迁移路径]
- 兼容性：[互操作问题]
- 学习成本：[转换成本]
```

### 4. 故障排除类问题（X出错了）
**模板结构：**
```
### 问题：[问题现象]的解决方法

**问题诊断**
- 症状描述：[具体表现]
- 可能原因：[潜在原因列表]
- 影响范围：[影响程度]

**解决步骤**
1. 基本信息收集：[收集什么信息]
2. 原因排查：[如何定位问题]
3. 解决方案：[具体修复方法]
4. 预防措施：[避免再次发生]

**应急方案**
- 临时解决：[临时应对方法]
- 回滚方案：[恢复原状方法]
- 专家求助：[何时寻求帮助]
```

## 模板使用说明
1. 根据问题类型选择合适的模板
2. 填充模板中的占位符内容
3. 确保回答的完整性和准确性
4. 记录本次问答到日志系统
