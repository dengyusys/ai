## 技能摘要
- **save-point**：记录项目进展并更新记忆系统，创建知识检查点
  - **用途**：任务完成时记录成果，长时间对话后总结进展，项目状态变化时保存快照
  - **激活方式**：用户明确要求或自动检测（任务完成、长时间对话、项目状态变化）
  - **更新文件**：每日笔记（memory/daily/YYYY-MM-DD.md）和长期记忆（memory/MEMORY.md）
  - **核心流程**：收集任务完成情况、关键成果、重要决策、遇到的问题、下一步计划

- **teaching-session-recorder**：记录教学对话到知识库
  - **用途**：捕获有价值的教学互动，构建个人教学知识库，特别适用于英语学习等场景
  - **激活方式**：用户明确要求（如"record chat"、"save conversation"、"记录对话"、"保存对话"）
  - **更新文件**：聊天记录（knowledge/chat/chat-YYYY-MM-DD.md）
  - **核心流程**：将教学对话整理为结构化的Markdown文档，包含时间戳、对话内容、教学要点
