# 个人知识管理系统 - CLAUDE.md

## 这是什么
用AI帮助管理知识的系统。

**核心**：`knowledge/` 目录

**怎么沟通**：
- 用中文，英文术语
- 基于知识库回答，说清楚来源
- 简单直接

**怎么引用**：
```
> 来自：[文件名]
> 引用：[内容]
> 相关：[其他知识]
```

## 知识在哪
所有文件都在 `knowledge/` 目录，没有子文件夹。

**知识种类**（只是分类）：
- 系统知识
- 临时笔记
- 参考资料
- 项目文档

## AI怎么做

### 回答问题
1. 听懂问题
2. 在 `knowledge/` 里找相关文件
3. 用找到的知识回答，说明来源

### 不同问题
- **问是什么**：找文件，引用
- **问怎么做**：分析，给方法
- **问怎么整理**：建议分类
- **问想法**：关联不同知识

### 管理知识
- **加新的**：联系已有知识
- **找旧的**：搜索，判断有用性
- **改旧的**：记录为什么改

### 重要原则
- 新知识要联系旧的
- 保持简单
